# Friendly Asset Manifest

Design assets generated with Recraft AI for the Friendly app.

## Directory Structure

### `/activities/`
Activity illustration assets for circle categories and event types.
- `activity-running.png` — Running/jogging illustration
- `activity-cooking.png` — Cooking/food illustration
- `activity-reading.png` — Book club/reading illustration
- `activity-gaming.png` — Board games illustration
- `activity-outdoors.png` — Outdoor/hiking illustration
- `activity-yoga.png` — Yoga/wellness illustration
- `activity-music.png` — Music/jam session illustration
- `activity-art.png` — Arts and crafts illustration
- `activity-volunteer.png` — Volunteering illustration
- `activity-social.png` — Social gathering illustration

## Asset Specifications

| Type | Size | Format | Notes |
|------|------|--------|-------|
| Activities | 512x512 | PNG | Transparent background |
| Avatars | 256x256 | PNG | Circular crop ready |
| Badges | 256x256 | PNG | Transparent background |
| Icons | Various | PNG | @1x, @2x, @3x variants needed |
| Medals | 256x256 | PNG | Transparent background |
| Onboarding | 750x750 | PNG | Transparent background |
| Season | 1125x600 | PNG | Full bleed backgrounds |
